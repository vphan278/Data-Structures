# Data Structures and Algorithms (C++)

This repository contains implementations of common data structures and algorithms written in C++.  
The goal of this project is to understand how these structures work internally by building them from scratch.

These implementations were developed while studying Data Structures and practicing algorithm design and analysis.

--------------------------------------------------

DATA STRUCTURES IMPLEMENTED

Linked Lists
- Singly Linked List
- Doubly Linked List
- Node insertion and deletion
- Pointer manipulation

Trees
- Binary Search Tree (BST)
- Tree traversal algorithms
  - In-order
  - Pre-order
  - Post-order

Heaps
- Min Heap
- Max Heap
- Heap insertion and removal

Hash Tables
- Hash functions
- Collision handling

Graphs
- Graph representations
- Breadth-First Search (BFS)
- Depth-First Search (DFS)

Queues
- Queue implementation using linked list

--------------------------------------------------

ALGORITHMS

Sorting Algorithms
- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort
- Quick Sort

Big-O Analysis
Examples and exercises analyzing algorithm time complexity and efficiency.
